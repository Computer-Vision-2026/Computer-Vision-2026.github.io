About the Course

Bridging Classic and Modern Vision

Machine learning has profoundly transformed computer vision, yet modern methods have deep roots in the field's history. This comprehensive lecture explores these foundations while fully incorporating the latest deep learning advances.

Taking a holistic approach that goes beyond machine learning, we address fundamental issues in the task of vision and the critical relationship of machine vision to human perception, covering a wide range of fundamental and modern computer vision topics.

Learning Objectives

By the end of this course, you will be able to:

  • Understand the historical foundations and modern advances in computer vision
  • Connect classical computer vision methods with contemporary deep learning approaches
  • Address fundamental issues in vision and human perception
  • Master next-generation topics including transformers and diffusion models
  • Apply computer vision techniques to real-world problems

Course Structure

Real-time Video Object Tracking

This project implements a state-of-the-art object tracking system that operates in real-time on video streams. By combining deep feature extraction with efficient tracking algorithms, we aim to build a robust system capable of tracking multiple objects simultaneously while maintaining high frame rates. The system will handle occlusions and scale variations common in real-world scenarios, with applications in surveillance, sports analytics, and autonomous systems.
